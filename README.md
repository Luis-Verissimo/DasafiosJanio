# 🚀 DesafiosJanio

## 📌 Sobre o Projeto

O **DesafiosJanio** é um repositório criado com o objetivo de promover a evolução técnica e profissional por meio de desafios práticos de programação.

A proposta central é comparar o desempenho, estilo e abordagem de **três programadores em cada linguagem**, sendo elas:

* Python 🐍
* C# 💻
* Kotlin 📱

Cada participante resolve os mesmos desafios, permitindo uma análise rica sobre diferentes formas de pensar, estruturar e otimizar soluções.

---

## 🎯 Objetivos

* Incentivar o estudo contínuo das linguagens propostas
* Desenvolver novas técnicas de resolução de problemas
* Explorar diferentes estratégias de mercado e boas práticas
* Estimular a troca de conhecimento entre programadores
* Melhorar competências profissionais de forma prática e comparativa

---

## 🧠 Como Funciona

* Cada desafio é proposto com um problema específico
* Três programadores por linguagem implementam suas soluções
* As soluções são comparadas considerando:

  * Performance ⚡
  * Legibilidade 📖
  * Escalabilidade 📈
  * Criatividade 💡
